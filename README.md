# UAV Forest Fire Management System

[cite_start]A comprehensive 3D GIS web-based system designed to manage Unmanned Aerial Vehicle (UAV) flight paths and detect forest fires in real-time[cite: 1862]. 

## 🌟 Key Features
- [cite_start]**3D GIS Mapping**: Real-time visualization of UAVs, terrain, and forest zones using CesiumJS[cite: 1930].
- [cite_start]**Automated Flight Planning**: Create, save, and execute automated 3D patrol routes via waypoints[cite: 1980, 2052].
- [cite_start]**Real-time Monitoring**: Track UAV telemetry data including altitude (MSL/AGL), battery level, and heading[cite: 2045].
- [cite_start]**AI Fire Detection**: Integrated YOLOv8 model for real-time fire and smoke object detection from UAV camera feeds[cite: 1948, 1951].
- [cite_start]**Alert System**: Instant visual and audio alerts when fire is detected[cite: 2083].

## 🛠️ Tech Stack
- [cite_start]**Frontend**: CesiumJS, HTML, JavaScript[cite: 1928, 3223].
- [cite_start]**Backend**: FastAPI (Python), SQLite[cite: 3223, 3248].
- [cite_start]**AI Model**: Ultralytics YOLOv8[cite: 1948].
